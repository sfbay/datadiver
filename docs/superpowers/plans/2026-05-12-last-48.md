# The Last 48 Implementation Plan

> **For agentic workers:** REQUIRED: Use superpowers:subagent-driven-development (if subagents available) or superpowers:executing-plans to implement this plan. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Replace the scanner-only `LiveFeeds` view at `/live-feeds` with **The Last 48** — a comprehensive map-based surface for SF's freshest civic data, with two modes (FLOW = animated event arrival, HOTSPOTS = z-score anomaly choropleth), a per-dataset polling engine respecting each source's natural refresh cadence, and a "quick peek → deep dive" detail-panel pattern that gateways into the dedicated dataset views.

**Architecture:** Single page at `/live-feeds` with URL-encoded mode (`?mode=flow|hotspots`). Central `useLast48Window` hook polls 5 live-tier datasets at their individual cadences, maintains an in-memory 48-hour sliding buffer keyed on `(datasetId, eventId)`, and exposes a normalized event stream. Both modes consume the same hook — toggling between them is a renderer swap, never a re-fetch. Kiosk mode (`?mode=kiosk`) is **deferred to a follow-up plan** per the "Live first → Kiosk layered on" sequencing.

**Tech Stack:** React 18, TypeScript, Vite, Mapbox GL JS v3, D3.js, Zustand, Tailwind v4, Socrata SODA API. No new dependencies.

**Verification model (modified TDD):** DataDiver has no unit-test infrastructure by convention. Every task verifies via (a) `npx tsc -b` for type safety, (b) targeted `curl` against the live Socrata API for data-engine logic, (c) dev-server visual smoke via Tarmac MCP for UI components, then (d) Vercel preview deploy at the PR boundary.

**Spec:** `docs/superpowers/specs/2026-05-12-last-48-design.md`

**Out of scope (defer to separate plans):**
- Phase 4 / KIOSK director — separate brainstorm + plan after Live ships and is calibrated against the user's 55" portrait monitor
- OmniSearch v2 integration on this view
- Multi-city support
- Server-side push / SSE

```typescript
// src/hooks/useTabVisibility.ts
//
// Returns `true` when the tab is visible, `false` when hidden.
// Used by usePollCadence to pause polling when the user has the tab
// in the background — saves Socrata quota and avoids stale data races
// on tab restoration.

import { useEffect, useState } from 'react'

export function useTabVisibility(): boolean {
  const [visible, setVisible] = useState(() =>
    typeof document !== 'undefined' ? document.visibilityState === 'visible' : true
  )

  useEffect(() => {
    if (typeof document === 'undefined') return
    const onChange = () => setVisible(document.visibilityState === 'visible')
    document.addEventListener('visibilitychange', onChange)
    return () => document.removeEventListener('visibilitychange', onChange)
  }, [])

  return visible
}
```

```typescript
// src/hooks/usePollCadence.ts
//
// Schedules a callback at a configurable cadence with:
//   - Random jitter (0-20% of interval) to avoid all datasets pinging
//     Socrata at the same second on a shared cadence
//   - Pause when tab is hidden (via useTabVisibility); on resume, fires
//     immediately once before resuming the schedule
//   - Exponential backoff on consecutive errors, capped at 30 min
//
// Caller is responsible for tracking the success/error of each call
// and notifying via the `onResult` callback so backoff can advance.

import { useEffect, useRef } from 'react'
import { useTabVisibility } from './useTabVisibility'

interface PollCadenceOpts {
  /** Base interval in milliseconds */
  intervalMs: number
  /** Maximum backoff ceiling (defaults to 30 min) */
  maxBackoffMs?: number
  /** Function to invoke on each tick */
  fetch: () => Promise<void>
  /** Optional debug label for console logs */
  label?: string
}

const DEFAULT_MAX_BACKOFF_MS = 30 * 60 * 1000

export function usePollCadence({
  intervalMs,
  maxBackoffMs = DEFAULT_MAX_BACKOFF_MS,
  fetch,
  label = 'poll',
}: PollCadenceOpts): void {
  const visible = useTabVisibility()
  const fetchRef = useRef(fetch)
  fetchRef.current = fetch
  const consecutiveErrorsRef = useRef(0)

  useEffect(() => {
    if (!visible) return

    let cancelled = false
    let timeoutId: ReturnType<typeof setTimeout> | undefined

    const tick = async () => {
      if (cancelled) return
      try {
        await fetchRef.current()
        consecutiveErrorsRef.current = 0
      } catch (e) {
        consecutiveErrorsRef.current += 1
        if (typeof console !== 'undefined') {
          console.warn(`[${label}] poll error #${consecutiveErrorsRef.current}`, e)
        }
      }
      if (cancelled) return

      // Compute next delay with jitter + backoff
      const backoffMultiplier = Math.min(
        Math.pow(2, consecutiveErrorsRef.current),
        maxBackoffMs / intervalMs
      )
      const baseDelay = intervalMs * backoffMultiplier
      const jitter = baseDelay * Math.random() * 0.2
      const nextDelay = Math.min(baseDelay + jitter, maxBackoffMs)

      timeoutId = setTimeout(tick, nextDelay)
    }

    // Fire immediately on mount / visibility return, then schedule next
    void tick()

    return () => {
      cancelled = true
      if (timeoutId !== undefined) clearTimeout(timeoutId)
    }
  }, [visible, intervalMs, maxBackoffMs, label])
}
```

```typescript
// src/utils/eventNormalization.ts
//
// Maps raw Socrata rows to NormalizedEvent. Each dataset has its own
// schema; the normalizer is dataset-specific. The mapper picks out the
// canonical fields (timestamp, location, callType, headline) for use
// in FLOW/HOTSPOTS rendering while preserving the original row in `raw`
// for the detail panel.

import type { DatasetId, NormalizedEvent } from '@/types/last48'

interface RawCoord {
  longitude?: string | number
  latitude?: string | number
}

/** Parse "POINT (lon lat)" Socrata strings to { lon, lat }, or undefined */
function parsePoint(p: unknown): { lon: number; lat: number } | undefined {
  if (typeof p !== 'string') return undefined
  const match = p.match(/POINT\s*\(\s*(-?\d+\.?\d*)\s+(-?\d+\.?\d*)\s*\)/i)
  if (!match) return undefined
  return { lon: parseFloat(match[1]), lat: parseFloat(match[2]) }
}

/** Pull a coord pair from either Socrata `point` shape or scattered fields */
function coords(row: Record<string, unknown>, fields: RawCoord = {}): { lon?: number; lat?: number } {
  const point = parsePoint(row.point) || parsePoint(row.case_location) || parsePoint(row.intersection)
  if (point) return { lon: point.lon, lat: point.lat }
  const lon = fields.longitude ?? row.longitude ?? row.long ?? row.lon
  const lat = fields.latitude ?? row.latitude ?? row.lat
  return {
    lon: lon != null ? parseFloat(String(lon)) : undefined,
    lat: lat != null ? parseFloat(String(lat)) : undefined,
  }
}

function parseTimestamp(s: unknown): { iso: string; ms: number } | null {
  if (typeof s !== 'string') return null
  const ms = Date.parse(s)
  if (isNaN(ms)) return null
  return { iso: s, ms }
}

export function normalizeEvent(
  datasetId: DatasetId,
  row: Record<string, unknown>
): NormalizedEvent | null {
  switch (datasetId) {
    case '911-realtime':
    case '911-historical': {
      const t = parseTimestamp(row.received_datetime)
      if (!t) return null
      const c = coords(row)
      return {
        id: `${datasetId}:${row.cad_number ?? row.dispatch_id ?? row.id}`,
        datasetId,
        timestamp: t.iso,
        receivedAt: t.ms,
        neighborhood: row.analysis_neighborhood as string | undefined,
        longitude: c.lon,
        latitude: c.lat,
        callType: row.call_type_final_desc as string | undefined,
        headline: (row.call_type_final_desc as string | undefined) ?? '911 dispatch',
        raw: row,
      }
    }

    case 'fire-ems-dispatch': {
      const t = parseTimestamp(row.received_dttm)
      if (!t) return null
      const c = coords(row)
      return {
        id: `fire-ems-dispatch:${row.call_number ?? row.unit_id}`,
        datasetId,
        timestamp: t.iso,
        receivedAt: t.ms,
        neighborhood: row.neighborhoods_analysis_boundaries as string | undefined,
        longitude: c.lon,
        latitude: c.lat,
        callType: row.call_type as string | undefined,
        headline: (row.call_type as string | undefined) ?? 'Fire/EMS dispatch',
        raw: row,
      }
    }

    case '311-cases': {
      const t = parseTimestamp(row.requested_datetime)
      if (!t) return null
      const c = coords(row)
      return {
        id: `311-cases:${row.service_request_id}`,
        datasetId,
        timestamp: t.iso,
        receivedAt: t.ms,
        neighborhood: row.neighborhoods_sffind_boundaries as string | undefined,
        longitude: c.lon,
        latitude: c.lat,
        callType: row.service_subtype as string | undefined,
        headline: (row.service_subtype as string | undefined) ?? (row.service_name as string | undefined) ?? '311 case',
        raw: row,
      }
    }

    case 'parking-revenue': {
      const t = parseTimestamp(row.session_start_dt)
      if (!t) return null
      const c = coords(row)
      return {
        id: `parking-revenue:${row.post_id ?? row.session_id ?? `${row.session_start_dt}-${row.meter_id}`}`,
        datasetId,
        timestamp: t.iso,
        receivedAt: t.ms,
        neighborhood: row.analysis_neighborhood as string | undefined,
        longitude: c.lon,
        latitude: c.lat,
        callType: row.payment_type as string | undefined,
        headline: row.street_name ? `Meter @ ${row.street_name}` : 'Parking session',
        raw: row,
      }
    }

    case 'police-incidents': {
      const t = parseTimestamp(row.incident_datetime)
      if (!t) return null
      const c = coords(row)
      return {
        id: `police-incidents:${row.incident_id ?? row.incident_number}`,
        datasetId,
        timestamp: t.iso,
        receivedAt: t.ms,
        neighborhood: row.analysis_neighborhood as string | undefined,
        longitude: c.lon,
        latitude: c.lat,
        callType: row.incident_category as string | undefined,
        headline: (row.incident_subcategory as string | undefined) ?? (row.incident_category as string | undefined) ?? 'Police incident',
        raw: row,
      }
    }
  }
}
```

```typescript
// src/hooks/useLast48Window.ts
//
// Central data engine for The Last 48. Polls each enabled dataset on
// its own cadence, maintains an in-memory sliding 48h buffer keyed on
// (datasetId, eventId), and exposes a normalized event stream + per-
// dataset freshness metrics.
//
// Architecture choices:
//   - useRef for the buffer (avoids re-rendering on every poll)
//   - useSyncExternalStore for stable snapshot subscription
//   - Per-dataset poll cadence via usePollCadence
//   - 48h eviction on each poll
//   - Deduplication via id Map; deduplicated events do NOT count as "new"
//
// Returns a stable result object that components destructure.

import { useEffect, useRef, useSyncExternalStore } from 'react'
import { fetchDataset } from '@/api/client'
import { useTabVisibility } from './useTabVisibility'
import { usePollCadence } from './usePollCadence'
import { normalizeEvent } from '@/utils/eventNormalization'
import type {
  DatasetId,
  EventTier,
  FreshnessMap,
  NormalizedEvent,
  DatasetFreshness,
} from '@/types/last48'
import { ALL_LAST48_DATASETS } from '@/types/last48'

// Per-dataset poll intervals (ms). Inferred from observed publish cadence;
// see spec "Data Sources & Latency Tiers" for the table.
const POLL_INTERVALS: Record<DatasetId, number> = {
  '911-realtime':       2 * 60 * 1000,
  'fire-ems-dispatch':  30 * 60 * 1000,
  '311-cases':          30 * 60 * 1000,
  '911-historical':     30 * 60 * 1000,
  'parking-revenue':    4 * 60 * 60 * 1000,
  'police-incidents':   6 * 60 * 60 * 1000,
}

// Map DatasetId → datasets.ts key (since the dataset registry uses different
// keys than our DatasetId union). Keep this aligned with src/api/datasets.ts.
const DATASET_REGISTRY_KEY: Record<DatasetId, string> = {
  '911-realtime':       'dispatch911Realtime',
  'fire-ems-dispatch':  'fireEMSDispatch',
  '311-cases':          'cases311',
  '911-historical':     'dispatch911',
  'parking-revenue':    'parkingMeterRevenue',
  'police-incidents':   'policeIncidents',
}

// Map DatasetId → date field for WHERE filtering.
const DATE_FIELD: Record<DatasetId, string> = {
  '911-realtime':       'received_datetime',
  'fire-ems-dispatch':  'received_dttm',
  '311-cases':          'requested_datetime',
  '911-historical':     'received_datetime',
  'parking-revenue':    'session_start_dt',
  'police-incidents':   'incident_datetime',
}

interface State {
  byId: Map<string, NormalizedEvent>
  freshness: FreshnessMap
  isPolling: Record<DatasetId, boolean>
  initialLoadComplete: boolean
}

export interface Last48WindowResult {
  events: NormalizedEvent[]                   // sorted newest-first
  byDataset: Record<DatasetId, NormalizedEvent[]>
  freshness: FreshnessMap
  isLoading: boolean
  isPolling: boolean
  refetch: () => void
}

interface UseLast48WindowOpts {
  datasets: DatasetId[]
  tier?: EventTier                            // optional, for default dataset selection
}

const WINDOW_MS = 48 * 60 * 60 * 1000

function emptyFreshness(): DatasetFreshness {
  return { rowsUpdatedAt: null, maxEventTime: null, eventLagMs: null, refreshLagMs: null, error: null }
}

function initialState(): State {
  const freshness = {} as FreshnessMap
  for (const id of ALL_LAST48_DATASETS) freshness[id] = emptyFreshness()
  const isPolling = {} as Record<DatasetId, boolean>
  for (const id of ALL_LAST48_DATASETS) isPolling[id] = false
  return { byId: new Map(), freshness, isPolling, initialLoadComplete: false }
}

export function useLast48Window(opts: UseLast48WindowOpts): Last48WindowResult {
  // External store with mutable ref + listeners
  const stateRef = useRef<State>(initialState())
  const listenersRef = useRef<Set<() => void>>(new Set())

  const notify = () => {
    for (const l of listenersRef.current) l()
  }

  // Subscribe via useSyncExternalStore
  const state = useSyncExternalStore(
    (cb) => {
      listenersRef.current.add(cb)
      return () => listenersRef.current.delete(cb)
    },
    () => stateRef.current,
    () => stateRef.current
  )

  // Fetcher for one dataset
  const buildFetcher = (datasetId: DatasetId) => async () => {
    const dateField = DATE_FIELD[datasetId]
    const registryKey = DATASET_REGISTRY_KEY[datasetId]
    const since = new Date(Date.now() - WINDOW_MS).toISOString().replace('Z', '')

    stateRef.current = {
      ...stateRef.current,
      isPolling: { ...stateRef.current.isPolling, [datasetId]: true },
    }
    notify()

    try {
      const rows = await fetchDataset<Record<string, unknown>>(registryKey, {
        $where: `${dateField} >= '${since}'`,
        $order: `${dateField} DESC`,
        $limit: 5000,
      })

      const next = new Map(stateRef.current.byId)
      const cutoff = Date.now() - WINDOW_MS
      // Evict old
      for (const [k, v] of next) {
        if (v.datasetId === datasetId && v.receivedAt < cutoff) next.delete(k)
      }
      // Add/replace
      let maxEvent = stateRef.current.freshness[datasetId].maxEventTime ?? 0
      for (const row of rows) {
        const norm = normalizeEvent(datasetId, row)
        if (!norm) continue
        next.set(norm.id, norm)
        if (norm.receivedAt > maxEvent) maxEvent = norm.receivedAt
      }

      // rowsUpdatedAt via dataset metadata is fetched separately — for now
      // we approximate by "now" since the API responded. Future: hit
      // /api/views/{id}.json in parallel for the real rowsUpdatedAt.
      const now = Date.now()
      const nextFreshness: DatasetFreshness = {
        rowsUpdatedAt: now,
        maxEventTime: maxEvent,
        eventLagMs: now - maxEvent,
        refreshLagMs: 0,
        error: null,
      }

      stateRef.current = {
        ...stateRef.current,
        byId: next,
        freshness: { ...stateRef.current.freshness, [datasetId]: nextFreshness },
        isPolling: { ...stateRef.current.isPolling, [datasetId]: false },
        initialLoadComplete: true,
      }
      notify()
    } catch (e) {
      const msg = e instanceof Error ? e.message : String(e)
      stateRef.current = {
        ...stateRef.current,
        freshness: {
          ...stateRef.current.freshness,
          [datasetId]: { ...stateRef.current.freshness[datasetId], error: msg },
        },
        isPolling: { ...stateRef.current.isPolling, [datasetId]: false },
      }
      notify()
      throw e
    }
  }

  // Schedule a polling cadence per dataset
  for (const datasetId of opts.datasets) {
    // eslint-disable-next-line react-hooks/rules-of-hooks
    usePollCadence({
      intervalMs: POLL_INTERVALS[datasetId],
      fetch: buildFetcher(datasetId),
      label: datasetId,
    })
  }

  // Derive return value from current state
  const events: NormalizedEvent[] = Array.from(state.byId.values()).sort(
    (a, b) => b.receivedAt - a.receivedAt
  )
  const byDataset = {} as Record<DatasetId, NormalizedEvent[]>
  for (const id of ALL_LAST48_DATASETS) byDataset[id] = []
  for (const ev of events) {
    if (byDataset[ev.datasetId]) byDataset[ev.datasetId].push(ev)
  }

  return {
    events,
    byDataset,
    freshness: state.freshness,
    isLoading: !state.initialLoadComplete,
    isPolling: Object.values(state.isPolling).some(Boolean),
    refetch: () => {
      // Trigger an immediate fetch on all enabled datasets
      for (const id of opts.datasets) void buildFetcher(id)()
    },
  }
}
```

```tsx
function formatLag(ms: number | null): string {
  if (ms == null) return '—'
  const sec = Math.floor(ms / 1000)
  if (sec < 90) return `${sec}s`
  const min = Math.floor(sec / 60)
  if (min < 90) return `${min}m`
  const h = Math.floor(min / 60)
  if (h < 48) return `${h}h`
  const d = Math.floor(h / 24)
  return `${d}d`
}

function lagColor(ms: number | null): string {
  if (ms == null) return 'text-paper-600'
  const minutes = ms / 60_000
  if (minutes < 30) return 'text-moss-600 dark:text-moss-400'         // < 30 min — fresh
  if (minutes < 360) return 'text-paper-700 dark:text-paper-400'      // < 6h — neutral
  if (minutes < 1440) return 'text-ochre-700 dark:text-ochre-500'     // < 24h — warning
  return 'text-brick-700 dark:text-brick-500'                          // >= 24h — concern
}

export default function FreshnessChipStrip({ freshness }: { freshness: FreshnessMap }) {

              <span className={lagColor(f.refreshLagMs)}>{formatLag(f.refreshLagMs)}</span>
            </span>

      <div className="flex flex-wrap items-center gap-x-3 gap-y-1">

        {present.map((id) => {
          const f = freshness[id]
          return (
            <span key={`lag-${id}`} className="flex items-baseline gap-1">
              <span className="text-paper-700 dark:text-paper-500">{DATASET_LABELS[id]}</span>
              <span className={lagColor(f.eventLagMs)}>{formatLag(f.eventLagMs)}</span>
```

```tsx
interface Props {
  map: MapboxMap | null
  events: NormalizedEvent[]
  onEventClick?: (event: NormalizedEvent) => void
}

export default function FlowMapLayer({ map, events, onEventClick }: Props) {
  const onClickRef = useRef(onEventClick)
  onClickRef.current = onEventClick

  // Build GeoJSON from events that have coordinates
  const geojson = useMemo(() => {
    const features = events
      .filter((e) => e.longitude != null && e.latitude != null)
      .map((e, i) => ({
        type: 'Feature' as const,
        id: i,
        properties: {
          id: e.id,
          datasetId: e.datasetId,
          color: COLORS[e.datasetId],
          age: Date.now() - e.receivedAt,
          headline: e.headline ?? '',
        },
        geometry: { type: 'Point' as const, coordinates: [e.longitude!, e.latitude!] },
      }))
    return { type: 'FeatureCollection' as const, features }
  }, [events])

  useMapLayer({
    map,
    sourceId: 'last48-flow-events',
    layerId: 'last48-flow-events-circles',
    source: { type: 'geojson', data: geojson },
    layer: {
      type: 'circle',
      paint: {
        'circle-color': ['get', 'color'],
        'circle-radius': [
          'interpolate', ['linear'], ['zoom'],
          10, 3,
          14, 6,
        ],
        // Fade older events to 40% opacity over the 48h window
        'circle-opacity': [
          'interpolate', ['linear'], ['get', 'age'],
          0, 1.0,
          172800000, 0.4,  // 48h
        ],
        'circle-stroke-color': '#1e140d',
        'circle-stroke-width': 0.5,
      },
    } as mapboxgl.AnyLayer,
  })

  // Click handler — wire once when the map is ready
  useEffect(() => {
    if (!map) return
    const handler = (e: mapboxgl.MapMouseEvent & { features?: mapboxgl.MapboxGeoJSONFeature[] }) => {
      const feature = e.features?.[0]
      if (!feature) return
      const id = feature.properties?.id as string | undefined
      if (!id) return
      const ev = events.find((x) => x.id === id)
      if (ev && onClickRef.current) onClickRef.current(ev)
    }
    map.on('click', 'last48-flow-events-circles', handler)
    map.on('mouseenter', 'last48-flow-events-circles', () => { map.getCanvas().style.cursor = 'pointer' })
    map.on('mouseleave', 'last48-flow-events-circles', () => { map.getCanvas().style.cursor = '' })
    return () => {
      map.off('click', 'last48-flow-events-circles', handler)
    }
  }, [map, events])

  return null
}
```

```typescript
// Computes per-(neighborhood × dataset) z-scores for HOTSPOTS mode.
//
// Baseline window: 12 weeks of non-overlapping 48h windows = 42 samples
// per (neighborhood, dataset). Fetched server-side via a single Socrata
// query per dataset that GROUP BY neighborhood and bucket events into
// 48h windows.
//
// Cached for the session — refreshes only when the hook is first
// instantiated (HOTSPOTS mode entry). Re-entry uses cache.

import { useEffect, useState } from 'react'
import { fetchDataset } from '@/api/client'
import type {
  AnomalyResult,
  DatasetId,
  NormalizedEvent,
} from '@/types/last48'

interface BaselineEntry {
  // Per-neighborhood: array of 42 historical 48h counts
  historicalCounts: Record<string, number[]>
}

interface CacheValue {
  baseline: Record<DatasetId, BaselineEntry>
  fetchedAt: number
}

const CACHE_TTL = 4 * 60 * 60 * 1000  // 4h — baseline shifts slowly
let cache: CacheValue | null = null

// Same mappings as useLast48Window
const DATASET_REGISTRY_KEY: Record<DatasetId, string> = {
  '911-realtime':       'dispatch911Realtime',
  'fire-ems-dispatch':  'fireEMSDispatch',
  '311-cases':          'cases311',
  '911-historical':     'dispatch911',
  'parking-revenue':    'parkingMeterRevenue',
  'police-incidents':   'policeIncidents',
}
const DATE_FIELD: Record<DatasetId, string> = {
  '911-realtime':       'received_datetime',
  'fire-ems-dispatch':  'received_dttm',
  '311-cases':          'requested_datetime',
  '911-historical':     'received_datetime',
  'parking-revenue':    'session_start_dt',
  'police-incidents':   'incident_datetime',
}
const NEIGHBORHOOD_FIELD: Record<DatasetId, string> = {
  '911-realtime':       'analysis_neighborhood',
  'fire-ems-dispatch':  'neighborhoods_analysis_boundaries',
  '311-cases':          'neighborhoods_sffind_boundaries',
  '911-historical':     'analysis_neighborhood',
  'parking-revenue':    'analysis_neighborhood',
  'police-incidents':   'analysis_neighborhood',
}

interface BaselineRow {
  neighborhood: string
  window_start: string  // 48h-bucket start (we'll bucket client-side from daily counts)
  cnt: string
}

async function fetchBaselineForDataset(datasetId: DatasetId): Promise<BaselineEntry> {
  const dateField = DATE_FIELD[datasetId]
  const nhField = NEIGHBORHOOD_FIELD[datasetId]
  const registryKey = DATASET_REGISTRY_KEY[datasetId]

  // Get daily counts for the last 12 weeks (84 days), grouped by neighborhood.
  // We'll bucket into 48h windows client-side.
  const since = new Date(Date.now() - 84 * 24 * 60 * 60 * 1000).toISOString().replace('Z', '')

  const rows = await fetchDataset<BaselineRow>(registryKey, {
    $select: `${nhField} as neighborhood, date_trunc_ymd(${dateField}) as window_start, COUNT(*) as cnt`,
    $where: `${dateField} >= '${since}' AND ${nhField} IS NOT NULL`,
    $group: `${nhField}, date_trunc_ymd(${dateField})`,
    $limit: 50000,
  })

  // Bucket daily counts into 48h windows per neighborhood
  const byNeighborhood: Record<string, Record<string, number>> = {}
  for (const r of rows) {
    if (!r.neighborhood) continue
    const dayMs = Date.parse(r.window_start)
    if (isNaN(dayMs)) continue
    // 48h bucket = floor(daysSinceEpoch / 2) * 2
    const days = Math.floor(dayMs / (24 * 60 * 60 * 1000))
    const bucket = Math.floor(days / 2) * 2
    const key = String(bucket)
    if (!byNeighborhood[r.neighborhood]) byNeighborhood[r.neighborhood] = {}
    byNeighborhood[r.neighborhood][key] = (byNeighborhood[r.neighborhood][key] ?? 0) + parseInt(r.cnt, 10)
  }

  // Convert to arrays of counts
  const historicalCounts: Record<string, number[]> = {}
  for (const [nh, buckets] of Object.entries(byNeighborhood)) {
    historicalCounts[nh] = Object.values(buckets)
  }

  return { historicalCounts }
}

function mean(xs: number[]): number {
  if (xs.length === 0) return 0
  return xs.reduce((a, b) => a + b, 0) / xs.length
}

function stdDev(xs: number[], m: number): number {
  if (xs.length < 2) return 0
  const variance = xs.reduce((s, x) => s + (x - m) ** 2, 0) / (xs.length - 1)
  return Math.sqrt(variance)
}

export interface UseAnomalyBaselineResult {
  anomalies: AnomalyResult[]
  isLoading: boolean
  error: string | null
}

export function useAnomalyBaseline(opts: {
  datasets: DatasetId[]
  currentEvents: NormalizedEvent[]
}): UseAnomalyBaselineResult {
  const [baseline, setBaseline] = useState<Record<DatasetId, BaselineEntry> | null>(null)
  const [isLoading, setIsLoading] = useState(true)
  const [error, setError] = useState<string | null>(null)

  useEffect(() => {
    if (cache && Date.now() - cache.fetchedAt < CACHE_TTL) {
      setBaseline(cache.baseline)
      setIsLoading(false)
      return
    }

    let cancelled = false
    setIsLoading(true)

    Promise.all(
      opts.datasets.map((id) => fetchBaselineForDataset(id).then((b) => [id, b] as const))
    )
      .then((entries) => {
        if (cancelled) return
        const b: Record<DatasetId, BaselineEntry> = {} as Record<DatasetId, BaselineEntry>
        for (const [id, entry] of entries) b[id] = entry
        cache = { baseline: b, fetchedAt: Date.now() }
        setBaseline(b)
        setIsLoading(false)
      })
      .catch((e) => {
        if (cancelled) return
        setError(e instanceof Error ? e.message : String(e))
        setIsLoading(false)
      })

    return () => { cancelled = true }
    // eslint-disable-next-line react-hooks/exhaustive-deps
  }, [opts.datasets.join(',')])

  // Compute current 48h counts per (neighborhood × dataset)
  const anomalies: AnomalyResult[] = []
  if (baseline) {
    const currentCounts: Record<DatasetId, Record<string, number>> = {} as Record<DatasetId, Record<string, number>>
    for (const id of opts.datasets) currentCounts[id] = {}
    for (const ev of opts.currentEvents) {
      if (!ev.neighborhood) continue
      if (!currentCounts[ev.datasetId]) continue
      currentCounts[ev.datasetId][ev.neighborhood] = (currentCounts[ev.datasetId][ev.neighborhood] ?? 0) + 1
    }

    for (const datasetId of opts.datasets) {
      const entry = baseline[datasetId]
      if (!entry) continue
      for (const [nh, history] of Object.entries(entry.historicalCounts)) {
        if (history.length < 5) continue  // not enough N for a defensible σ
        const m = mean(history)
        const sd = stdDev(history, m)
        if (sd === 0) continue  // can't divide
        const cur = currentCounts[datasetId][nh] ?? 0
        const z = (cur - m) / sd
        anomalies.push({
          neighborhood: nh,
          datasetId,
          count48h: cur,
          baselineMean: m,
          baselineSd: sd,
          zScore: z,
        })
      }
    }
  }

  return { anomalies, isLoading, error }
}
```

```tsx
// src/views/Last48/modes/HotspotsChoropleth.tsx

import { useMemo, useEffect, useRef } from 'react'
import type { Map as MapboxMap } from 'mapbox-gl'
import { useMapLayer } from '@/hooks/useMapLayer'
import { useNeighborhoodBoundaries } from '@/hooks/useNeighborhoodBoundaries'
import type { AnomalyResult } from '@/types/last48'

interface Props {
  map: MapboxMap | null
  /** Combined z-score per neighborhood (averaged across selected datasets) */
  combinedAnomalies: Record<string, number>
  onNeighborhoodClick?: (neighborhood: string) => void
}

export default function HotspotsChoropleth({ map, combinedAnomalies, onNeighborhoodClick }: Props) {
  const { data: boundaries } = useNeighborhoodBoundaries()
  const onClickRef = useRef(onNeighborhoodClick)
  onClickRef.current = onNeighborhoodClick

  const geojson = useMemo(() => {
    if (!boundaries) return { type: 'FeatureCollection' as const, features: [] }
    return {
      type: 'FeatureCollection' as const,
      features: boundaries.features.map((f) => {
        const nhName = (f.properties as { name?: string; nhood?: string })?.name
          ?? (f.properties as { nhood?: string })?.nhood
          ?? ''
        const z = combinedAnomalies[nhName] ?? 0
        return {
          ...f,
          properties: { ...f.properties, name: nhName, zScore: z },
        }
      }),
    }
  }, [boundaries, combinedAnomalies])

  useMapLayer({
    map,
    sourceId: 'last48-hotspots-source',
    layerId: 'last48-hotspots-fill',
    source: { type: 'geojson', data: geojson },
    layer: {
      type: 'fill',
      paint: {
        'fill-color': [
          'case',
          ['<', ['get', 'zScore'], -2],   '#a8926a',   // strongly below baseline — paper
          ['<', ['get', 'zScore'], -0.5], '#bda37d',   // below baseline — paper-light
          ['<', ['get', 'zScore'],  0.5], 'transparent', // within ±0.5σ — no color
          ['<', ['get', 'zScore'],  1.0], '#d4a435',   // ochre
          ['<', ['get', 'zScore'],  2.0], '#d47149',   // terracotta
          '#963e30',                                     // brick (>+2σ)
        ],
        'fill-opacity': [
          'case',
          ['<', ['abs', ['get', 'zScore']], 0.5], 0,
          0.65,
        ],
      },
    } as mapboxgl.AnyLayer,
  })

  // Outline layer
  useMapLayer({
    map,
    sourceId: 'last48-hotspots-source',
    layerId: 'last48-hotspots-outline',
    source: { type: 'geojson', data: geojson },
    layer: {
      type: 'line',
      paint: {
        'line-color': [
          'case',
          ['<', ['get', 'zScore'], -0.5], '#a8926a',
          ['>', ['get', 'zScore'],  0.5], '#1e140d',
          'transparent',
        ],
        'line-width': 1,
        'line-dasharray': [
          'case',
          ['<', ['get', 'zScore'], -0.5], ['literal', [2, 2]],
          ['literal', [1, 0]],
        ] as any,
      },
    } as mapboxgl.AnyLayer,
  })

  // Click handler
  useEffect(() => {
    if (!map) return
    const handler = (e: mapboxgl.MapMouseEvent & { features?: mapboxgl.MapboxGeoJSONFeature[] }) => {
      const feature = e.features?.[0]
      const nh = feature?.properties?.name as string | undefined
      if (nh && onClickRef.current) onClickRef.current(nh)
    }
    map.on('click', 'last48-hotspots-fill', handler)
    map.on('mouseenter', 'last48-hotspots-fill', () => { map.getCanvas().style.cursor = 'pointer' })
    map.on('mouseleave', 'last48-hotspots-fill', () => { map.getCanvas().style.cursor = '' })
    return () => { map.off('click', 'last48-hotspots-fill', handler) }
  }, [map])

  return null
}
```

```tsx
import type { AnomalyResult } from '@/types/last48'

function zColor(z: number): string {
  if (z >= 2)   return '#963e30'
  if (z >= 1)   return '#d47149'
  if (z >= 0.5) return '#d4a435'
  if (z <= -0.5) return '#a8926a'

interface Props {
  /** Combined per-neighborhood z-score (averaged across selected datasets) */
  combinedAnomalies: Record<string, number>
  selectedNeighborhood?: string
  onSelect: (neighborhood: string) => void
}

export default function AnomalyRail({ combinedAnomalies, selectedNeighborhood, onSelect }: Props) {
  const entries = Object.entries(combinedAnomalies)
    .filter(([, z]) => Math.abs(z) >= 0.5)
    .sort(([, a], [, b]) => Math.abs(b) - Math.abs(a))
    .slice(0, 30)

  return (
    <aside className="w-[clamp(180px,16vw,260px)] border-l border-paper-200/40 dark:border-espresso-700 bg-paper-50/40 dark:bg-espresso-950/60 flex flex-col">

        <h2 className="font-mono text-[10px] tracking-widest text-paper-600 dark:text-paper-500">
          STANDS OUT
        </h2>
        <p className="font-mono text-[9px] text-paper-500 dark:text-paper-600 mt-0.5">
          vs typical 48h window · 12-week baseline
        </p>
      </div>

      <div className="flex-1 overflow-y-auto px-2 py-2 flex flex-col gap-1">
        {entries.map(([nh, z], idx) => {
          const isSel = nh === selectedNeighborhood
          return (
            <button
              key={nh}

              onClick={() => onSelect(nh)}

                  ? 'bg-ochre-500/20 ring-1 ring-ochre-500'
                  : 'hover:bg-paper-200/40 dark:hover:bg-espresso-800/60'}
              `}
            >
              <span className="flex items-baseline gap-1.5 min-w-0">
                <span className="text-paper-500 dark:text-paper-600 tabular-nums">{idx + 1}·</span>
                <span className="text-paper-800 dark:text-paper-300 truncate">{nh}</span>
              </span>
              <span className="tabular-nums" style={{ color: zColor(z) }}>
                {z >= 0 ? '+' : ''}{z.toFixed(1)}σ
              </span>
            </button>
          )
        })}
        {entries.length === 0 && (
          <div className="text-paper-500 dark:text-paper-600 text-center italic py-6">
            no notable anomalies right now
          </div>
        )}
      </div>
    </aside>
  )
}
```

```tsx
// src/views/Last48/modes/HotspotsMode.tsx

import { useMemo, useState } from 'react'
import MapView, { type MapHandle } from '@/components/maps/MapView'
import { useAnomalyBaseline } from '@/hooks/useAnomalyBaseline'
import type { Last48WindowResult } from '@/hooks/useLast48Window'
import type { DatasetId } from '@/types/last48'
import HotspotsChoropleth from './HotspotsChoropleth'
import AnomalyRail from './AnomalyRail'
import Last48NeighborhoodPeek from '../detail/Last48NeighborhoodPeek'

interface Props {
  window48: Last48WindowResult
  datasets: DatasetId[]
}

export default function HotspotsMode({ window48, datasets }: Props) {
  const [mapHandle, setMapHandle] = useState<MapHandle | null>(null)
  const [selectedNh, setSelectedNh] = useState<string | null>(null)

  const { anomalies, isLoading } = useAnomalyBaseline({
    datasets,
    currentEvents: window48.events.filter((e) => datasets.includes(e.datasetId)),
  })

  // Combine per-(nh × dataset) z-scores into a single per-nh number,
  // averaged across selected datasets. (Inverse-variance weighting is
  // documented as a future enhancement in the spec.)
  const combined = useMemo(() => {
    const sums: Record<string, { total: number; n: number }> = {}
    for (const a of anomalies) {
      if (!sums[a.neighborhood]) sums[a.neighborhood] = { total: 0, n: 0 }
      sums[a.neighborhood].total += a.zScore
      sums[a.neighborhood].n += 1
    }
    const result: Record<string, number> = {}
    for (const [nh, s] of Object.entries(sums)) {
      result[nh] = s.n > 0 ? s.total / s.n : 0
    }
    return result
  }, [anomalies])

  return (
    <div className="absolute inset-0 flex">
      <div className="flex-1 relative">
        <MapView onMapReady={setMapHandle}>
          <HotspotsChoropleth
            map={mapHandle?.getMap() ?? null}
            combinedAnomalies={combined}
            onNeighborhoodClick={setSelectedNh}
          />
        </MapView>
        {isLoading && (
          <div className="absolute top-3 left-3 font-mono text-[10px] text-paper-500 bg-espresso-900/70 px-2 py-1 rounded">
            computing 12-week baseline…
          </div>
        )}
      </div>

      <AnomalyRail
        combinedAnomalies={combined}
        selectedNeighborhood={selectedNh ?? undefined}
        onSelect={(nh) => {
          setSelectedNh(nh)
          // Optional fly-to could go here if we have the polygon centroid
        }}
      />

      {selectedNh && (
        <Last48NeighborhoodPeek
          neighborhood={selectedNh}
          anomalies={anomalies.filter((a) => a.neighborhood === selectedNh)}
          events={window48.events.filter((e) => e.neighborhood === selectedNh && datasets.includes(e.datasetId))}
          onClose={() => setSelectedNh(null)}
        />
      )}
    </div>
  )
}
```

```tsx
// HOTSPOTS click target — diagnoses an anomaly rather than just naming
// it. Shows per-dataset z-score breakdown, top contributing events,
// reuses NeighborhoodCensusContext for demographic framing.

import type { AnomalyResult, NormalizedEvent, DatasetId } from '@/types/last48'

const DATASET_LABELS: Record<DatasetId, string> = {
  '911-realtime':      '911',
  'fire-ems-dispatch': 'Fire/EMS',
  '311-cases':         '311',
  '911-historical':    '911 Hist',
  'parking-revenue':   'Parking',
  'police-incidents':  'Police',
}

function zColor(z: number): string {
  if (z >= 1.5)  return '#963e30'
  if (z >= 0.5)  return '#d47149'
  if (z >= -0.5) return '#7a5f42'
  if (z >= -1.5) return '#a8926a'
  return '#a8926a'
}

interface Props {
  neighborhood: string
  anomalies: AnomalyResult[]
  events: NormalizedEvent[]
  onClose: () => void
}

export default function Last48NeighborhoodPeek({ neighborhood, anomalies, events, onClose }: Props) {
  const overallZ = anomalies.length > 0
    ? anomalies.reduce((s, a) => s + a.zScore, 0) / anomalies.length
    : 0

  const isAbove = overallZ >= 0
  const summaryLine = isAbove ? 'unusual activity in the last 48 hours' : 'below baseline activity in the last 48 hours'

  // Compute max |z| for the bar width scaling
  const maxAbsZ = Math.max(2, ...anomalies.map((a) => Math.abs(a.zScore)))

  // Top contributing events (5 most recent in this neighborhood)
  const topEvents = events.slice(0, 5)

  return (
    <aside className="absolute top-0 right-0 w-[clamp(280px,28vw,400px)] h-full bg-paper-50 dark:bg-espresso-900 border-l border-paper-300 dark:border-espresso-700 z-30 flex flex-col">

        onClick={onClose}
        className="self-end p-3 text-paper-500 hover:text-paper-300 text-lg leading-none"
        aria-label="Close panel"
      >
        ✕
      </button>

      <div className="flex-1 overflow-y-auto px-5 pb-4 flex flex-col gap-4">
        {/* Eyebrow */}
        <div className="font-mono text-[10px] tracking-widest text-paper-600 dark:text-paper-500">

          <p className="font-mono text-[11px] text-paper-700 dark:text-paper-400 mt-1 italic">
            {summaryLine}
          </p>
        </div>

        {/* Per-dataset breakdown */}
        <section>
          <h3 className="font-mono text-[10px] tracking-wider text-paper-600 dark:text-paper-500 mb-2">
            PER-DATASET BREAKDOWN
          </h3>
          <ul className="flex flex-col gap-1.5 font-mono text-[10px]">
            {anomalies.map((a) => {
              const widthPct = Math.min(100, (Math.abs(a.zScore) / maxAbsZ) * 100)
              return (
                <li key={a.datasetId} className="flex items-center gap-2">
                  <span className="w-16 text-paper-700 dark:text-paper-400 truncate">{DATASET_LABELS[a.datasetId as DatasetId] ?? a.datasetId}</span>

                    <span
                      className="block h-full rounded"
                      style={{ width: `${widthPct}%`, backgroundColor: zColor(a.zScore) }}
                    />
                  </span>
                </li>
              )
            })}
          </ul>
        </section>

        {/* Top contributing events */}
        <section>
          <h3 className="font-mono text-[10px] tracking-wider text-paper-600 dark:text-paper-500 mb-2">
            TOP CONTRIBUTING EVENTS
          </h3>
          <ul className="flex flex-col gap-1 font-mono text-[10px]">
            {topEvents.map((ev) => (
```

```tsx
// src/views/Last48/modes/FlowScrubber.tsx
//
// Time scrubber sitting along the bottom edge of the FLOW map. Tracks
// a time cursor over the 48h window; in timelapse mode, the cursor
// sweeps from window-start to NOW at variable speed.

import { useEffect, useRef, useState } from 'react'

const WINDOW_MS = 48 * 60 * 60 * 1000

interface Props {
  /** Cursor position as ms-from-window-start (0 = 48h ago, WINDOW_MS = NOW) */
  cursor: number
  /** Setter for parent to update cursor (drag) */
  onCursorChange: (next: number) => void
  /** Whether the scrubber is currently playing back */
  isPlaying: boolean
  onTogglePlay: () => void
  /** Current playback speed multiplier (1, 2, 4, or 8) */
  speed: number
  onCycleSpeed: () => void
  /** Whether timelapse should loop on completion */
  loop: boolean
  onToggleLoop: () => void
}

const TICKS = ['SUN', 'MON', 'TUE', 'NOW']

export default function FlowScrubber({
  cursor, onCursorChange, isPlaying, onTogglePlay, speed, onCycleSpeed, loop, onToggleLoop,
}: Props) {
  const trackRef = useRef<HTMLDivElement>(null)
  const [dragging, setDragging] = useState(false)

  const cursorPct = Math.max(0, Math.min(100, (cursor / WINDOW_MS) * 100))

  // Drag handlers
  useEffect(() => {
    if (!dragging) return
    const onMove = (e: MouseEvent) => {
      if (!trackRef.current) return
      const rect = trackRef.current.getBoundingClientRect()
      const x = e.clientX - rect.left
      const pct = Math.max(0, Math.min(1, x / rect.width))
      onCursorChange(pct * WINDOW_MS)
    }
    const onUp = () => setDragging(false)
    window.addEventListener('mousemove', onMove)
    window.addEventListener('mouseup', onUp, { once: true })
    return () => window.removeEventListener('mousemove', onMove)
  }, [dragging, onCursorChange])

  return (
    <div className="absolute bottom-3 left-3 right-3 bg-espresso-900/85 backdrop-blur-sm rounded-md px-3 py-2 z-15">
      {/* Track */}
      <div
        ref={trackRef}
        onMouseDown={(e) => {
          setDragging(true)
          // Set cursor on first click
          const rect = e.currentTarget.getBoundingClientRect()
          const x = e.clientX - rect.left
          const pct = Math.max(0, Math.min(1, x / rect.width))
          onCursorChange(pct * WINDOW_MS)
        }}
        className="relative h-3 cursor-pointer"
      >
        <div className="absolute inset-x-0 top-1 h-1 bg-paper-200/20 rounded" />
        <div
          className="absolute top-1 h-1 rounded bg-gradient-to-r from-paper-300/30 to-ochre-500/50"
          style={{ width: `${cursorPct}%` }}
        />
        <div
          className="absolute top-0 w-3 h-3 -translate-x-1/2 bg-ochre-500 rounded-full shadow-md"
          style={{ left: `${cursorPct}%`, boxShadow: '0 0 8px #d4a435' }}
        />
      </div>

      {/* Tick labels */}
      <div className="flex justify-between font-mono text-[8px] text-paper-600 mt-1">
        {TICKS.map((t) => <span key={t}>{t}</span>)}
      </div>

      {/* Controls */}
      <div className="flex items-center gap-3 mt-2 font-mono text-[10px]">
        <button
          type="button"
          onClick={onTogglePlay}
          className="px-2 py-0.5 rounded bg-ochre-500/20 hover:bg-ochre-500/40 text-paper-300 tracking-wider"
          aria-label={isPlaying ? 'Pause' : 'Play'}
        >
          {isPlaying ? '⏸' : '▶▶'}
        </button>
        <button
          type="button"
          onClick={onCycleSpeed}
          className="text-paper-500 hover:text-paper-300 tracking-wider"
          aria-label="Cycle speed"
        >
          {speed}×
        </button>
        <label className="flex items-center gap-1.5 cursor-pointer text-paper-500 hover:text-paper-300 ml-auto">
          <input type="checkbox" checked={loop} onChange={onToggleLoop} className="accent-ochre-500" />
          loop
        </label>
      </div>
    </div>
  )
}
```

```tsx
const WINDOW_MS = 48 * 60 * 60 * 1000

// Scrubber state. cursor = ms from window start (WINDOW_MS = NOW)
const [cursor, setCursor] = useState(WINDOW_MS)
const [isPlaying, setIsPlaying] = useState(false)
const [speed, setSpeed] = useState(1)
const [loop, setLoop] = useState(false)

// Playback animation
useEffect(() => {
  if (!isPlaying) return
  let raf: number
  let lastT = performance.now()
  const tick = (now: number) => {
    const dt = now - lastT
    lastT = now
    setCursor((prev) => {
      const next = prev + dt * speed
      if (next >= WINDOW_MS) {
        if (loop) return 0
        setIsPlaying(false)
        return WINDOW_MS
      }
      return next
    })
    raf = requestAnimationFrame(tick)
  }
  raf = requestAnimationFrame(tick)
  return () => cancelAnimationFrame(raf)
}, [isPlaying, speed, loop])

// Filter events visible at cursor — events occurring within a 15-min
// window centered on the cursor. When cursor === WINDOW_MS, show events
// up to NOW unconditionally (the default live-edge state).
const cursorEpochMs = Date.now() - (WINDOW_MS - cursor)
const isLiveEdge = cursor >= WINDOW_MS - 1000  // within last second of slider
const visibleEvents = isLiveEdge
  ? window48.events.filter((e) => datasets.includes(e.datasetId))
  : window48.events.filter((e) =>
      datasets.includes(e.datasetId) &&
      Math.abs(e.receivedAt - cursorEpochMs) <= 15 * 60 * 1000
    )

const cycleSpeed = () => {
  const cycle = [1, 2, 4, 8]
  const idx = cycle.indexOf(speed)
  setSpeed(cycle[(idx + 1) % cycle.length])
}
```
