Detail Panel: Fire Incident Enrichment

When a user clicks a fire dispatch point, the panel cross-references the Fire Incidents dataset to show outcome data below the existing response timeline.

A

Inline Expansion

Fire outcome data appears as a new section below the existing response timeline in the same panel. Collapsible sections: "Fire Outcome", "Cause & Origin", "Detection Systems".

Tabbed Panel

The detail panel gets two tabs: "Response" (existing timeline) and "Fire Report" (outcome, cause, detection). User switches between them.
